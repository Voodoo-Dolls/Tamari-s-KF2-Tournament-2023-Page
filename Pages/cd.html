ABOUT CD

Controlled Difficulty (CD) is a mod for KF2 originally created by blackout that allows for adjusting the game’s difficulty on a much more in-depth level than is normally possible.


Traditionally, CD has been used to create an experience that is equivalent to “harder than Hell on Earth” difficulty, however it also excels at providing a consistent experience.

This makes it ideal for challenge runs, speed runs, and tournaments like this one.


Here are some examples of the things CD allows us to achieve:


  • Static ZED health values
  • Static ZED spawn rate
  • Fleshpounds and Quarter Pounds never spawn enraged
  • All waves have the same number of ZEDs
  • All waves have the same composition of ZEDs
  • All waves spawn ZEDs in the same order
  • and a lot more, but these are the important ones..

In general, CD can be used to set these parameters to static values and even set them far outside of their default maximums, which allows for maximum flexibility and control over the pacing and intensity of the game.

This results in a more difficult, yet consistent, experience that demands a higher degree of teamwork, coordination, and skill. For many players, CD represents the ideal KF2 cooperative experience and scratches the itch, so to speak, for those players who want just a little bit more than what the game offers in its standard set of difficulties & modes.

CD & THIS TOURNAMENT

This tournament uses CD as a base in order to create as fair and equal of an environment as possible for all players involved. Without it, many of the dynamic variables of the vanilla game have a strong potential of interfering with the results and creating scenarios where a player's knowledge and mechanical skill are is adequately represented by the match results.


CD will be used in this tournament to create a difficulty that is slightly beyond that of Hell on Earth.
